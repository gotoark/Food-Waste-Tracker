# Screenshots Directory

This directory contains screenshots of the Food Waste Tracker application.

## How to Add Screenshots

1. **Take screenshots** of the running application at http://localhost:5173
2. **Save them** with the following names:
   - `dashboard.png` - Full dashboard view
   - `entry-form.png` - Entry form section
   - `statistics.png` - Statistics panel
   - `charts.png` - All three charts (pie, bar, line)
   - `entry-list.png` - Waste entry list with data
   - `mobile.png` - Mobile responsive view

3. **Recommended screenshot dimensions**:
   - Desktop views: 1920x1080 or 1440x900
   - Mobile view: 375x812 (iPhone size)

4. **Tips for great screenshots**:
   - Use the seed data utility at `/seed.html` to populate with sample data
   - Capture the app with diverse data showing all features
   - Ensure good lighting/contrast for visibility
   - Show interactive elements in action (hover states, etc.)

### Recommended Tools

**macOS:**
- QuickTime Player (built-in, free)
- OBS Studio (free, open source)
- Screen Studio (paid, professional)

**Windows:**
- OBS Studio (free, open source)
- Xbox Game Bar (built-in, Win+G)
- Camtasia (paid)

**Cross-platform:**
- Loom (free tier available)
- ShareX (Windows, free)

### Video Recording Tips

1. **Prepare the app**:
   - Seed with sample data using `/seed.html`
   - Clear browser console
   - Set browser zoom to 100%
   - Use a clean browser profile (no extensions visible)

2. **Recording settings**:
   - Resolution: 1920x1080 (1080p) or 1280x720 (720p)
   - Frame rate: 30 fps minimum, 60 fps preferred
   - Duration: 1-3 minutes (keep it concise)
   - Format: MP4 (H.264 codec for best compatibility)

3. **Demo flow** (suggested):
   - Show the empty dashboard (2-3 seconds)
   - Add 2-3 waste entries with different categories
   - Show statistics updating in real-time
   - Scroll through the charts
   - Demonstrate deleting an entry
   - Show mobile responsive view (optional)

4. **Upload options**:
   - **GitHub**: Drag video into a GitHub issue/PR comment, copy the URL
   - **YouTube**: Upload as unlisted, copy the video ID
   - **Vimeo**: Upload and get the embed link

### Quick Recording with QuickTime (macOS)

```bash
# Open QuickTime Player
# File > New Screen Recording
# Select area or full screen
# Click record button
# Stop when done (Cmd+Control+Esc)
# File > Export As > 1080p
```

### Quick Recording with OBS Studio

1. Add "Display Capture" source
2. Set output to 1920x1080
3. Start recording
4. Stop and find video in Videos folder
